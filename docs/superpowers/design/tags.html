Get rid of them, too

Hit Edit in the tag picker and it flips to Manage: rename a tag (the count tells you how many sessions it touches) or delete it with the trash. Deleting a tag just removes it everywhere — the sessions stay.

Because a novel or a thesis can sprout dozens of tags, this is where you prune. The picker's search box keeps the everyday list short even when the full set is long.

Plain, not painted

Tag chips are just text. Projects keep the one subtle colour cue; tags stay quiet so a busy row still reads.

Built for many

Searchable picker + counts. Works the same for four dev tags or forty scene tags.

Yours to shape

Starter set to begin, then create, rename, and delete freely. Right-click a session, or tag a whole selection at once.
