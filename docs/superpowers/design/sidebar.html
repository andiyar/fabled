Select many, act once

Hit Select (top-right) or ⌘-click rows — checkboxes appear, and an action bar slides in: Pin, Tag…, Archive, more under ⋯.

This is also where tags plug in — select a handful, tag them all at once. That's the natural bridge to the next screen we design.
